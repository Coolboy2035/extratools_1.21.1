# 🌍 世界生成优化说明文档

## 📋 优化概述
优化了银矿石的世界生成系统，使其更加平衡、真实和有趣。

---

## ⛏️ 银矿石生成配置

### **三种矿脉类型**

#### 1️⃣ **小型银矿脉** (Small Vein)
- **矿脉大小**: 4个方块
- **生成频率**: 每区块 12 次尝试 ⭐⭐⭐⭐⭐ (最常见)
- **生成高度**: Y -32 到 Y 64 (均匀分布)
- **特点**: 最容易找到，适合新手前期挖掘

#### 2️⃣ **普通银矿脉** (Normal Vein)
- **矿脉大小**: 8个方块
- **生成频率**: 每区块 8 次尝试 ⭐⭐⭐ (常见)
- **生成高度**: Y -64 到 Y 32 (三角分布，峰值在 Y -16)
- **特点**: 主要在地下深处，类似铁矿的分布

#### 3️⃣ **大型银矿脉** (Large Vein)
- **矿脉大小**: 12个方块
- **生成频率**: 平均每 16 个区块 1 次 ⭐ (稀有)
- **生成高度**: Y -64 到 Y 0 (均匀分布，仅深层)
- **特点**: 罕见但收益高，需要深度探索

## 📊 与原版矿石对比

| 矿石类型 | 相似的原版矿石 | 稀有度 |
|---------|-------------|--------|
| 小型银矿脉 | 煤矿 | 非常常见 |
| 普通银矿脉 | 铁矿 | 常见 |
| 大型银矿脉 | 钻石矿 | 稀有 |

## 🎮 玩家体验优化

### **优势**
1. ✅ **渐进式体验**: 
   - 浅层有少量矿石供新手使用
   - 深层有更多资源奖励探索

2. ✅ **平衡的资源分布**:
   - 不会过于稀有导致挫败感
   - 也不会过于常见失去价值感

3. ✅ **多样化挖矿**:
   - 小矿脉适合快速采集
   - 大矿脉适合集中挖掘

4. ✅ **深层探索激励**:
   - 大型矿脉只在深层生成
   - 鼓励玩家探索基岩附近区域

## 🔧 技术实现

### **生成层级**
```
Y 384  ┐
       │  
Y 64   ├─ 小型矿脉结束
       │  
Y 32   ├─ 普通矿脉结束 (峰值在 Y -16)
       │  
Y 0    ├─ 大型矿脉结束
       │  
Y -32  ├─ 小型矿脉开始
       │  
Y -64  └─ 普通矿脉和大型矿脉开始 (基岩层附近)
```

### **生成类型**
- **三角分布**: 普通矿脉在中心高度更集中
- **均匀分布**: 小型和大型矿脉在范围内平均分布

### **可生成位置**
- ✅ 石头 (Stone)
- ✅ 深板岩 (Deepslate)
- ✅ 所有主世界生物群系

## 🎲 掉落物配置

### **银矿石掉落**
- 使用精准采集: 掉落矿石方块
- 普通挖掘: 掉落 **7-9 个银粒**
- 时运附魔: 增加掉落数量

### **经验值**
- 挖掘银矿石不掉落经验 (配置为 0)
- 可以通过修改 `ModBlocks.java` 中的 `ConstantInt.of(0)` 来调整

## 🚀 下一步操作

### **重新生成数据**
运行以下命令生成更新后的世界生成数据：
```bash
./gradlew runData
```

### **测试世界生成**
1. 启动游戏: `./gradlew runClient`
2. 创建新世界进行测试
3. 使用 `/locate` 命令或挖掘来验证矿石分布

### **调整参数**
如果需要调整生成频率，修改以下文件：
- `ModPlacedFeatures.java` - 调整生成次数和高度
- `ModConfiguredFeatures.java` - 调整矿脉大小
