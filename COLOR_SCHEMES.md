## ⚙️ 速度参数说明

```java
// 速度参数对比
getRedGoldColor(offset, 0.5f)  // 慢速：2秒完成一个周期
getRedGoldColor(offset, 1.0f)  // 标准：1秒完成一个周期
getRedGoldColor(offset, 2.0f)  // 快速：0.5秒完成一个周期
getRedGoldColor(offset, 5.0f)  // 极快：0.2秒完成一个周期
```

**推荐设置**：
- **物品栏显示**：1.0f（标准速度，不会太快导致眼花）
- **世界渲染**：0.8f（稍慢，便于远距离观看）
- **特殊效果**：2.0f（快速流动，吸引注意力）

## 🎨 自定义颜色方案

### **创建你自己的颜色方案**

在 `ColorCycleHelper.java` 中添加新方法：

```java
/**
 * 蓝白渐变（可用于和平主题）
 */
public static int getBlueWhiteColor(float offset, float speed) {
    long time = System.currentTimeMillis();
    float wave = (float) Math.sin((time * speed / 1000.0f + offset) * Math.PI);
    
    int red = (int) (100 + (155 * ((wave + 1) / 2)));
    int green = (int) (150 + (105 * ((wave + 1) / 2)));
    int blue = 255;
    
    return FastColor.ARGB32.color(255, red, green, blue);
}
```
